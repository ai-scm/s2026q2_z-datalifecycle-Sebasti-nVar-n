# Etapa 4: Análisis y Visualización

Esta etapa cierra el ciclo de transformación, convirtiendo datos brutos en insights que impulsan la toma de decisiones estratégicas en la organización.

## Puntos clave:
* **Niveles de análisis:**
    * **Descriptivo:** ¿Qué pasó? (Dashboards y reportes históricos).
    * **Diagnóstico:** ¿Por qué pasó? (Análisis de tendencias y correlaciones).
    * **Predictivo:** ¿Qué pasará? (Uso de Machine Learning para proyecciones).
    * **Prescriptivo:** ¿Qué debemos hacer? (Optimización y automatización de decisiones).
* **Herramientas AWS:**
    * **Amazon QuickSight:** BI nativo en la nube para visualizaciones interactivas.
    * **Amazon Athena:** Análisis SQL sobre datos en S3 sin necesidad de servidores.
    * **Amazon SageMaker:** Plataforma para el desarrollo y despliegue de modelos de IA/ML.
* **Democratización:** Facilitar el acceso a los datos mediante el concepto de *Self-Service BI*, permitiendo a los usuarios de negocio explorar la información de forma autónoma.
